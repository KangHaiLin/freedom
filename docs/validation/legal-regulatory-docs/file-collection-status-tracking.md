## 状态代码说明

### 收集状态
- **待分配**：文件已识别，尚未分配责任人
- **已分配**：已分配责任人，尚未开始收集
- **收集中**：责任人正在收集文件
- **已收集**：文件已获取，待质量检查
- **质量检查中**：正在进行格式和内容质量检查
- **已通过**：文件通过质量检查，可投入使用
- **有问题**：收集过程遇到问题，需要协调解决
- **已归档**：文件已整理归档到正确位置

### 优先级代码
- **P0**：紧急，必须今天完成，影响会议准备
- **P1**：高，需要在截止日期前完成，影响关键路径
- **P2**：中，重要但有一定缓冲时间
- **P3**：低，可选或补充材料
