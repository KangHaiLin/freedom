## 文件格式要求

### 电子版格式
1. **首选格式**：PDF（可搜索、带书签）
2. **替代格式**：Word文档（如无PDF）
3. **网页存档**：如只有网页版，保存为PDF或MHT格式
4. **命名规范**：`[文件ID]_[文件简称]_[版本日期].pdf`
   - 示例：`LR-001_证券法_20200301.pdf`

### 质量要求
1. **完整性**：完整版本，无缺失章节
2. **清晰度**：文字清晰可读
3. **可搜索**：支持文本搜索
4. **书签**：如有PDF，应有目录书签
5. **版本**：明确标注版本号和生效日期

## 文件使用说明

### 会议材料包准备
1. **精简版**：为专家会议准备关键条款摘要（5-10页）
2. **完整版**：完整的法律法规文件包（备用）
3. **关键条款索引**：标注与项目相关的具体条款

### 文件分发
1. **会议前2天**（3月16日前）：发送精简版给参会人员
2. **会议现场**：准备完整版纸质或电子版备查
3. **保密要求**：所有文件按内部机密处理
