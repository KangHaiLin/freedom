## 会议内容记录

### 一、开场与项目背景介绍（王项目）
1. 欢迎所有参会人员，强调项目重要性：本项目是公司级重点项目，关系到未来3年量化业务的核心竞争力
2. 介绍项目背景：现有系统已无法支撑业务增长，迫切需要新一代高性能量化交易平台
3. 通报前期工作：需求分析、技术验证、可行性评审全部完成，项目正式进入开发阶段
4. 介绍参会人员，对各部门的支持表示感谢

### 二、项目总体介绍（王项目）
1. **项目目标**：
   - 业务目标：支持100+策略并行，研发效率提升50%，交易延迟毫秒级
   - 技术目标：回测性能≤180秒，写入性能≥10万条/秒，99.9%可用性
   - 上线时间：2026年9月30日正式上线

2. **项目范围**：
   - 包含数据采集、回测引擎、交易引擎、风控系统、可视化平台等6大模块
   - 排除策略研发、交易通道等已有业务

3. **成功标准**：
   - 按时上线，功能性能全部达标
   - 业务部门验收通过
   - 稳定运行30天无重大故障
   - 成本控制在预算范围内

### 三、技术方案与架构介绍（钱技术）
1. **技术选型**：
   - 混合数据库架构：InfluxDB（实时数据）+ ClickHouse集群（分析）+ TimescaleDB（业务数据）
   - 技术栈：Python后端 + Vue前端 + Kafka消息队列 + Docker容器化部署
   - 架构优势：高性能、高可用、易扩展、易维护

2. **可行性验证结果**：
   - 基准测试验证技术方案完全可行
   - 3节点ClickHouse集群写入16.8万条/秒，远超目标
   - 所有性能目标通过优化均可实现，无重大技术风险

3. **技术难点与应对**：
   - 高性能回测引擎：采用并行计算+内存映射技术
   - 低延迟交易：异步IO + 高速缓存
   - 大数据处理：分布式架构 + 列式存储

### 四、项目计划与里程碑（王项目）
1. **总体周期**：6.5个月（3月20日 - 9月30日）
2. **关键里程碑**：
   | 里程碑 | 时间 | 交付物 |
   |---------|------|--------|
   | 设计完成 | 4月30日 | 需求、架构设计文档 |
   | 功能完成 | 7月15日 | 可运行核心系统 |
   | 测试完成 | 8月31日 | 验收通过 |
   | 正式上线 | 9月30日 | 生产环境运行 |

3. **迭代开发模式**：
   - 两周一个迭代，共14个迭代
   - 每两周交付可用功能，快速反馈，快速调整
   - 优先开发核心功能，尽早可用

4. **预算说明**：
   - 总预算440-600万，已获批
   - 预留10%应急预算
   - 人力配置10人核心团队，资源已落实

### 五、团队分工与管理制度（王项目）
1. **团队介绍**：
   - 核心团队10人，平均行业经验5年以上，技术实力雄厚
   - 业务团队3人全程支持需求和验收
   - 各部门指定接口人，支持项目推进

2. **分工职责**：
   - 详细分工已在《团队职责说明书》中明确，会后分发
   - 设立RACI责任矩阵，权责清晰

3. **管理制度**：
   - 每日站会、每周周会、每两周迭代评审
   - 严格变更管理，需求变更必须评估影响
   - 代码审核、自动化测试，保证质量
   - 问题24小时响应，重大问题立即上报

### 六、领导致辞与动员
#### 张总（技术总监）讲话

   - 效率优先：快速迭代，尽早交付价值
   - 团队协作：跨部门密切配合，形成合力
3. 承诺公司层面会全力支持项目，解决资源问题，预祝项目圆满成功

#### 刘总（量化投资部总经理）讲话
1. 代表业务部门对技术团队的前期工作表示感谢，对技术方案充满信心
2. 业务团队会全力配合需求确认、测试和验收
3. 期望新系统能大幅提升策略研发效率，为公司创造更大收益
4. 鼓励团队攻坚克难，期待9月系统上线

### 七、问答与交流
参会人员提出的问题和解答：
1. **问**：项目是否支持北向交易等新业务场景？
   **答**：架构设计已考虑扩展性，后续可快速支持新业务场景。

2. **问**：数据安全和合规性如何保障？
   **答**：系统严格按照证券行业监管要求设计，数据加密、权限控制、操作审计等功能已纳入需求。

3. **问**：现有策略如何迁移到新系统？
   **答**：会提供迁移工具和指导，业务团队配合测试，逐步迁移，保证平滑过渡。

4. **问**：项目有没有什么风险需要业务部门配合？
   **答**：请业务部门及时确认需求，配合测试，有问题及时反馈，共同保证项目成功。
