## 注意事项

### 时间管理
1. 所有会议准时开始，准时结束
2. 专题评审前15分钟为准备时间
3. 会议记录员提前5分钟入场准备
4. 技术问题提前测试会议设备

### 参与要求
1. 评审委员需全程参与分配的任务
2. 会前完成预审和相关材料阅读
3. 会议中积极参与讨论
4. 按时提交评审反馈和报告

### 沟通渠道
1. **正式沟通**：会议、邮件、正式报告
2. **日常沟通**：企业微信/钉钉群
3. **问题跟踪**：JIRA/问题跟踪表
4. **文档共享**：Confluence/共享网盘

### 应急计划
1. **人员缺席**：提前通知，安排代理人
2. **技术故障**：备用会议链接，电话接入
3. **时间冲突**：提前调整，优先保证核心会议
4. **进度延误**：增加会议频率，延长工作时间
